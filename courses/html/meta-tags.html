META TAGS & SEO

seo-optimized.html 

<!DOCTYPE html>
<html lang="en">
<head>
  <!-- Primary Meta Tags -->
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  
  <!-- SEO Meta Tags -->
  <title>Web Dev School - Learn HTML, CSS & JavaScript</title>
  <meta name="description" content="Free web development tutorials for beginners. Learn HTML, CSS and JavaScript through interactive examples and projects.">
  <meta name="keywords" content="HTML, CSS, JavaScript, web development, learn to code">
  <meta name="author" content="Web Dev School">
  
  <!-- Open Graph / Social Media -->
  <meta property="og:title" content="Web Dev School - Learn to Code">
  <meta property="og:description" content="Free web development tutorials for beginners.">
  <meta property="og:type" content="website">
  <meta property="og:url" content="https://webdev.school">
  <meta property="og:image" content="https://webdev.school/images/og-preview.jpg">
  
  <!-- Twitter Card -->
  <meta name="twitter:card" content="summary_large_image">
  <meta name="twitter:creator" content="@webdevschool">
  
  <!-- Favicons -->
  <link rel="icon" href="/favicon.ico" sizes="any">
  <link rel="icon" href="/icon.svg" type="image/svg+xml">
  <link rel="apple-touch-icon" href="/apple-touch-icon.png">
  <link rel="manifest" href="/site.webmanifest">
  
  <!-- Canonical URL -->
  <link rel="canonical" href="https://webdev.school/html-tutorials">
  
  <!-- Preload Critical Resources -->
  <link rel="preload" href="/css/main.css" as="style">
  <link rel="preload" href="/js/main.js" as="script">
</head>
<body>
  <!-- Page content would go here -->
  <h1>Welcome to Web Dev School!</h1>
  <p>Check the head section for SEO meta tags</p>
</body>
</html>

Meta Tag Categories

      
      

        
1. Essential Meta Tags

        
Critical for all websites:

        

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
        

      

      
      

        
2. Social Media Meta

        
Control how content appears when shared:

        

<meta property="og:title" content="Page Title">
<meta property="og:image" content="image.jpg">
<meta name="twitter:card" content="summary_large_image">
        

      

      
      

        
3. Performance Tags

        
Improve loading speed:

        

<link rel="preload" href="style.css" as="style">
<link rel="preconnect" href="https://fonts.gstatic.com">
<link rel="dns-prefetch" href="//cdn.example.com">
