HTML BASICS

basic-document.html 

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Page</title>
  <style>
    body { font-family: Arial; text-align: center; }
    h1 { color: #6e45e2; }
  </style>
</head>
<body>
  <h1>Hello World!</h1>
  <p>Edit this code to see changes!</p>
</body>
</html>

The HTML Skeleton

1. DOCTYPE Declaration

Tells browsers this is an HTML5 document.

<!DOCTYPE html>

2. Root Element

Wraps all content. Always use lang attribute.

<html lang="en"></html>
