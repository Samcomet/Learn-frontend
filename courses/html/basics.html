What is HTML?

HTML is the skeleton of every website. It defines the structure and content.

üçé Analogy: If a website were a house:
- HTML = Walls and structure
- CSS = Paint and decorations
- JavaScript = Electricity and interactivity

1. The Basic Structure

Every HTML document needs these essential parts:

<!DOCTYPE html> <html> <head> <!-- Invisible settings go here --> </head> <body> <!-- Visible content goes here --> </body> </html>
‚úì DOCTYPE - Tells browsers this is HTML5
‚úì html - Wraps your entire page
‚úì head - Contains page settings (invisible)
‚úì body - Contains visible content

2. Common Tags You'll Use

<h1>Main Heading</h1> <!-- Biggest heading --> <h2>Subheading</h2> <!-- Smaller heading --> <p>Paragraph text.</p> <!-- Normal text --> <a href="#">Link</a> <!-- Clickable link --> <img src="photo.jpg"> <!-- Shows an image --> <button>Click Me</button> <!-- Interactive button -->
