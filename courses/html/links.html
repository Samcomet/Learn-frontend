Understanding Hyperlinks

Links are the fundamental building blocks of the web, allowing navigation between pages.

üîó Analogy: Think of links like doors between rooms in a house - each door leads to a different space.
links-page.html 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Links Demonstration</title>
  <style>
    body { font-family: Arial; padding: 2rem; }
    .link-example { margin-bottom: 1.5rem; padding: 1rem; border-left: 4px solid #6e45e2; }
  </style>
</head>
<body>
  <h1>HTML Links Guide</h1>
  
  <div class="link-example">
    <h2>1. Basic Link</h2>
    <a href="https://webdev.school">Visit Web Dev School</a>
  </div>
  
  <div class="link-example">
    <h2>2. Link with Target</h2>
    <a href="https://webdev.school" target="_blank" rel="noopener noreferrer">
      Open in New Tab
    </a>
  </div>
  
  <div class="link-example">
    <h2>3. Email Link</h2>
    <a href="mailto:contact@webdev.school">Email Us</a>
  </div>
  
  <div class="link-example">
    <h2>4. Anchor Link</h2>
    <a href="#section-about">Jump to About Section</a>
    <div id="section-about" style="height: 800px; margin-top: 2rem;">
      <h3>About Section</h3>
      <p>You've reached the anchor point!</p>
    </div>
  </div>
</body>
</html>
