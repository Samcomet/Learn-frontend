HTML TABLES

Organizing data in rows and columns

What Are HTML Tables?

Tables let you display information in a grid format, like:

  • Price comparisons
  • Class schedules
  • Sports standings
üìä Analogy: Think of tables like spreadsheets you create in Excel, but for websites!

Every table has 4 key parts:

1. Table Head

Column titles (Name, Age, etc.)

2. Table Body

Main content/data

3. Table Foot

Summary/totals

class-schedule.html 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Class Schedule</title>
  <style>
    body {
      font-family: Arial;
      padding: 2rem;
    }
    table {
      width: 100%;
      border-collapse: collapse;
    }
    th, td {
      padding: 1rem;
      text-align: left;
      border: 1px solid #ddd;
    }
    th {
      background-color: #6e45e2;
      color: white;
    }
    tr:nth-child(even) {
      background-color: #f2f2f2;
    }
    caption {
      font-weight: bold;
      margin-bottom: 1rem;
    }
  </style>
</head>
<body>
  <h1>My Class Schedule</h1>
  
  <table>
    <caption>Spring 2023 Schedule</caption>
    <thead>
      <tr>
        <th>Day</th>
        <th>Time</th>
        <th>Class</th>
        <th>Room</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Monday</td>
        <td>9:00 AM</td>
        <td>Web Development</td>
        <td>Lab 205</td>
      </tr>
      <tr>
        <td>Wednesday</td>
        <td>11:00 AM</td>
        <td>Graphic Design</td>
        <td>Room 101</td>
      </tr>
    </tbody>
    <tfoot>
      <tr>
        <td colspan="3">Total Classes:</td>
        <td>2</td>
      </tr>
    </tfoot>
  </table>
</body>
</html>

1. Table Building Blocks

<table> <caption>Table Title</caption> <thead> <!-- Column headers --> <tr> <!-- Table row --> <th>Header Cell</th> </tr> </thead> <tbody> <!-- Main content --> <tr> <td>Data Cell</td> </tr> </tbody> </table>
Every table needs these structural elements.

2. Spanning Cells

<td colspan="2">Spans 2 columns</td> <td rowspan="3">Spans 3 rows</td>
Use these when you need cells to merge across columns or rows.

3. Accessibility Tips

<th scope="col">Name</th> <!-- Column header --> <th scope="row">Total:</th> <!-- Row header -->
Helps screen readers understand table structure.
