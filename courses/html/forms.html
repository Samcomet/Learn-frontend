HTML FORMS

How to collect information from visitors

What Are Forms?

Forms are how websites collect information - like when you:

  • Sign up for a website
  • Leave a comment
  • Make a purchase
üìù Analogy: Think of forms like paper forms you fill out at the doctor's office, but digital!

Every form has 3 key parts:

1. Input fields (where you type)
2. Labels (tells you what to enter)
3. Submit button (sends your answers)

1. Form Building Blocks

<form> <!-- The form container --> <label for="id">Description:</label> <input type="text" id="id"> <button type="submit">Submit</button> </form>
Every form needs these three parts working together.

2. Common Input Types

<input type="text"> Regular text (names, etc.)
<input type="email"> Validates email format
<input type="password"> Hides what you type
<textarea></textarea> Multi-line text (comments)

3. Help Users Get It Right

<input required> <!-- Can't be left empty --> <input minlength="8"> <!-- At least 8 characters --> <input placeholder="Hint"> <!-- Example text -->
These help users fill out forms correctly on the first try.
