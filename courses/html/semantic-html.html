Semantic Elements

      
      

        
1. Structural Tags

        
Replace <div> with meaningful elements:

        

<header> <nav> <main>
<section> <article> <footer>
        

      

      
      

        
2. Landmark Roles

        
Screen readers use these for navigation:

        

<main role="main">
<nav aria-label="Primary navigation">
        

      

      
      

        
3. Text Semantics

        
Proper content tagging:

        

<time datetime="2023-01-01">New Year</time>
<address>Contact info</address>
